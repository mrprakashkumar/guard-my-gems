# 🚨 AI Scam Detector

An AI-powered web application that detects scam messages, emails, and screenshots using advanced multimodal reasoning with Google Gemini.

---

## 🧠 Problem Statement

Online scams are rapidly increasing through:

* WhatsApp messages
* Fake emails
* Fraudulent links
* Impersonation (banks, government, OTP scams)

Most users cannot easily identify whether a message is genuine or malicious.

---

## 💡 Solution

AI Scam Detector analyzes text and images to:

* Detect scam patterns
* Provide a scam probability score
* Explain why the content is suspicious
* Suggest safe actions to the user

---

## 🚀 Features

* 🔍 Text-based scam detection
* 🖼️ Screenshot/Image analysis (multimodal AI)
* 📊 Scam Score (0–100)
* 🧾 Detailed reasoning (why it's a scam)
* 🛡️ Safety recommendations
* ⚡ Fast and responsive UI

---

## 🧠 AI Integration

Powered by **Google Gemini API**:

* Natural language understanding
* Context-aware reasoning
* Multimodal (image + text) analysis

## ⚙️ Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/anurps07/guard-my-gems.git
cd guard-my-gems
```

### 2. Install dependencies

```bash
npm install
```

### 3. Create `.env` file

```env
VITE_GEMINI_API_KEY=your_api_key_here
```

### 4. Run the project

```bash
npm run dev
```

---

## 📸 Usage

1. Enter a message or upload a screenshot
2. Click **Analyze**
3. View:

   * Scam Score
   * Verdict (Safe / Suspicious / Scam)
   * Explanation
   * Suggested Action

---
